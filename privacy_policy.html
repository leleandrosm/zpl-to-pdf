Privacy Policy
ZPL to PDF - PRO

This document explains what data this Chrome extension accesses, how it is used, and your rights as a user.

Last updated: March 2026 Extension version: 2.0 Developer: Leandro Martins

01 — Overview

🔒

Short version: This extension does not collect, sell, share, or transmit any personal information to the developer or third parties. ZPL content is only sent to the Labelary API when you explicitly choose the Online conversion engine. All other data stays entirely on your device.

ZPL to PDF - PRO is a browser extension that converts ZPL (Zebra Programming Language) label files into PDF documents. It provides two independent conversion engines: an Online engine that uses the Labelary public API, and an Offline engine that processes everything locally using WebAssembly (WASM) with no network communication.

03 — Online Engine and Third-Party API

⚠️

When you use the Online (⚡ Labelary) engine, the content of your ZPL file is transmitted over HTTPS to api.labelary.com, a third-party public API. This transmission is required to generate the PDF preview and final document.

What this means for you

Labelary is a public ZPL rendering service. The extension sends only the ZPL source code — no file name, no user identity, no metadata — via an HTTPS POST request. The extension does not control how Labelary processes or retains that data. You can review Labelary's terms at labelary.com.

Choosing the Offline engine

If your ZPL files contain sensitive or proprietary data, use the Offline (📦 WASM) engine. No data is transmitted. All processing happens locally in your browser using a WebAssembly renderer.

✅

The Offline engine provides complete privacy: ZPL content never leaves your device and is discarded from memory as soon as the PDF is generated.

04 — Chrome Permissions Used

PermissionWhy it is needed
storage Saves user preferences (theme, language, engine choice) and recent PDF history locally on your device. No data leaves the device via this permission.
offscreen Required by Chrome Manifest V3 to run the WASM-based offline renderer in a background document without blocking the UI.
https://api.labelary.com/* Allows network requests to the Labelary API. Only used when the Online engine is selected. Scoped strictly to this single domain.

The extension requests no other permissions. It cannot access your browsing history, cookies, tabs, camera, microphone, or any website content.

How to clear all extension data

Chrome → Settings → Privacy and security → Site data and permissions → search for the extension → Delete. Uninstalling the extension also removes all stored data.

06 — Children's Privacy

This extension is designed for professional use in logistics, warehousing, and manufacturing environments. It is not directed at children under the age of 13. We do not knowingly collect any information from children.
